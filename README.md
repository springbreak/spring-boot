# learn-sprin-boot

### Run

requires `gradle` 2.1+

`gradle clean build bootrun`

if you want a `hot-swap` like feature, use [spring-loaded](https://github.com/spring-projects/spring-loaded)

```
java -javaagent:<pathTo>/springloaded-{VERSION}.jar -noverify
```
